# API Overview

Base URL: http://localhost:5000/api

- Auth
  - POST /auth/register
  - POST /auth/login
- Jobs
  - GET /jobs
  - POST /jobs
  - GET /jobs/:id
  - PATCH /jobs/:id
  - DELETE /jobs/:id
- Applications
  - POST /applications
  - GET /applications
- Saved Jobs
  - POST /saved
  - DELETE /saved/:id
- Analytics
  - GET /analytics
